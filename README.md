
# DRAGON403 Portal (React + Vite)

Sovereign DFIR portal UI with:
- Multi-language (ar/en/fr/tr/fa/es/zh) + RTL support
- Dark/Light/Auto themes
- Chat with file attachments, persistent memory, voice (STT/TTS)
- Export (JSON/MD/TXT/PDF via print)
- Screen & Audio recording
- RBAC (admin/analyst/viewer)

## Quick Start
```bash
npm i
npm run dev
```

## Build
```bash
npm run build
npm run preview
```
